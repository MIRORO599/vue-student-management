# 学生管理システム 基本設計書

## 1. システム概要

本システムは、学生情報を管理するためのWebアプリケーションである。
Vue 3、TypeScript、Vite を使用してフロントエンド画面を作成する。

本システムでは、学生情報の一覧表示、検索、新規登録、更新、削除を行う。
今回はフロントエンドのみで作成し、データは画面内の配列で管理する。

## 2. 対象範囲

対象範囲は以下の通りとする。

* 学生一覧表示
* 学生名検索
* 学生新規登録
* 学生情報更新
* 学生情報削除
* 入力チェック

バックエンドAPI連携、DB登録、ログイン機能は対象外とする。

## 3. 使用技術

| 項目      | 内容                    |
| ------- | --------------------- |
| フロントエンド | Vue 3                 |
| 言語      | TypeScript            |
| ビルドツール  | Vite                  |
| 画面構成    | Single File Component |

## 4. 画面一覧

| 画面ID   | 画面名      | 概要                       |
| ------ | -------- | ------------------------ |
| STU001 | 学生一覧画面   | 学生情報の一覧表示、検索、登録、更新、削除を行う |
| STU002 | 学生登録フォーム | 学生情報を新規登録する              |
| STU003 | 学生更新フォーム | 既存の学生情報を更新する             |

登録フォームと更新フォームは同一画面内で表示する。
更新時は選択された学生情報をフォームに表示する。

## 5. 機能一覧

| 機能ID | 機能名    | 処理概要                  |
| ---- | ------ | --------------------- |
| F001 | 学生一覧表示 | 学生情報をテーブル形式で表示する      |
| F002 | 学生名検索  | 入力された学生名を条件に学生情報を絞り込む |
| F003 | 学生登録   | 入力された内容を学生一覧に追加する     |
| F004 | 学生更新   | 選択された学生情報を入力内容で更新する   |
| F005 | 学生削除   | 選択された学生情報を一覧から削除する    |
| F006 | 入力チェック | 必須項目、数値範囲をチェックする      |

## 6. 一覧表示項目

| No | 項目名  | 内容           |
| -- | ---- | ------------ |
| 1  | 学生ID | 学生を一意に識別するID |
| 2  | 学生名  | 学生の氏名        |
| 3  | 誕生日  | 学生の誕生日       |
| 4  | 年齢   | 学生の年齢        |
| 5  | 成績   | 学生の成績        |
| 6  | クラス  | 所属クラス        |
| 7  | 先生名  | クラスに紐づく先生名   |
| 8  | 操作   | 更新、削除ボタン     |

## 7. 入力項目

| No | 項目名 | 型   | 必須 | 備考           |
| -- | --- | --- | -- | ------------ |
| 1  | 学生名 | 文字列 | 必須 | 最大50文字       |
| 2  | 誕生日 | 日付  | 必須 | yyyy-MM-dd形式 |
| 3  | 年齢  | 数値  | 必須 | 0以上150以下     |
| 4  | 成績  | 数値  | 必須 | 0以上100以下     |
| 5  | クラス | 数値  | 必須 | 選択式          |

## 8. 入力チェック

| 項目名 | チェック内容     | エラーメッセージ              |
| --- | ---------- | --------------------- |
| 学生名 | 未入力チェック    | 学生名を入力してください。         |
| 誕生日 | 未入力チェック    | 誕生日を入力してください。         |
| 年齢  | 未入力、範囲チェック | 年齢は0以上150以下で入力してください。 |
| 成績  | 未入力、範囲チェック | 成績は0以上100以下で入力してください。 |
| クラス | 未選択チェック    | クラスを選択してください。         |

## 9. 備考

今回は学習用のため、バックエンドAPIは使用しない。
実案件でAPI連携する場合は、一覧取得、登録、更新、削除の各APIを呼び出す構成に変更する。
