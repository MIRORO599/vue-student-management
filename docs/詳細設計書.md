# 学生管理システム 詳細設計書

## 1. コンポーネント構成

| ファイル                            | 内容                              |
| ------------------------------- | ------------------------------- |
| src/main.ts                     | Vueアプリケーションを作成し、App.vueを画面に表示する |
| src/App.vue                     | StudentManagementコンポーネントを読み込む   |
| src/views/StudentManagement.vue | 学生管理画面の表示と処理を行う                 |

## 2. 型定義

### Student

学生情報を表す型。

| 項目           | 型               | 内容   |
| ------------ | --------------- | ---- |
| studentId    | number          | 学生ID |
| studentName  | string          | 学生名  |
| birthday     | string          | 誕生日  |
| age          | number または null | 年齢   |
| score        | number または null | 成績   |
| studentClass | number          | クラス  |

### Teacher

先生情報を表す型。

| 項目           | 型      | 内容  |
| ------------ | ------ | --- |
| studentClass | number | クラス |
| teacherName  | string | 先生名 |

## 3. 変数定義

| 変数名           | 型                | 内容         |
| ------------- | ---------------- | ---------- |
| searchName    | ref string       | 検索条件の学生名   |
| isFormVisible | ref boolean      | フォーム表示状態   |
| isEditMode    | ref boolean      | 登録、更新モード判定 |
| errorMessages | ref string配列     | 入力チェックエラー  |
| students      | ref Student配列    | 学生一覧       |
| teachers      | ref Teacher配列    | 先生一覧       |
| form          | reactive Student | 登録、更新フォーム  |

## 4. 算出プロパティ

### filteredStudents

学生名検索の条件に一致する学生一覧を返却する。
searchName が空の場合は、全件を返却する。
searchName に値がある場合は、学生名に検索文字列を含むデータのみ返却する。

## 5. 関数定義

| 関数名              | 処理内容                               |
| ---------------- | ---------------------------------- |
| openCreateForm   | 登録モードでフォームを表示し、フォームを初期化する          |
| openEditForm     | 更新モードでフォームを表示し、選択された学生情報をフォームへ設定する |
| saveStudent      | 入力チェックを行い、登録または更新処理を実行する           |
| createStudent    | 新しい学生IDを採番し、学生一覧へ追加する              |
| updateStudent    | 学生IDが一致する学生情報を更新する                 |
| deleteStudent    | 確認ダイアログ表示後、対象学生を削除する               |
| clearSearch      | 検索条件をクリアする                         |
| closeForm        | フォームを非表示にし、エラーと入力値を初期化する           |
| resetForm        | フォームの値を初期化する                       |
| getNextStudentId | 学生一覧から次の学生IDを取得する                  |
| getTeacherName   | クラスに対応する先生名を取得する                   |
| validateForm     | 入力チェックを行い、エラーメッセージを返却する            |

## 6. 処理フロー

### 6.1 初期表示

1. App.vue から StudentManagement.vue を読み込む。
2. students に定義された初期データを表示する。
3. teachers に定義された先生情報をもとに先生名を表示する。

### 6.2 学生名検索

1. 学生名検索欄に文字を入力する。
2. searchName の値が更新される。
3. filteredStudents が再計算される。
4. 条件に一致する学生情報を一覧に表示する。

### 6.3 学生登録

1. 新規登録ボタンを押下する。
2. 登録モードでフォームを表示する。
3. 学生情報を入力する。
4. 保存ボタンを押下する。
5. validateForm で入力チェックを行う。
6. エラーがある場合はエラーメッセージを表示する。
7. エラーがない場合は createStudent を実行する。
8. 学生一覧に新しい学生情報を追加する。
9. フォームを閉じる。

### 6.4 学生更新

1. 一覧の更新ボタンを押下する。
2. 更新モードでフォームを表示する。
3. 選択された学生情報をフォームに設定する。
4. 内容を修正する。
5. 保存ボタンを押下する。
6. validateForm で入力チェックを行う。
7. エラーがある場合はエラーメッセージを表示する。
8. エラーがない場合は updateStudent を実行する。
9. 学生一覧の対象データを更新する。
10. フォームを閉じる。

### 6.5 学生削除

1. 一覧の削除ボタンを押下する。
2. 確認ダイアログを表示する。
3. キャンセルの場合は処理を終了する。
4. OKの場合は対象学生を一覧から削除する。

## 7. 入力チェック詳細

| 項目名 | チェック内容 | 条件                  |
| --- | ------ | ------------------- |
| 学生名 | 必須チェック | 空文字の場合エラー           |
| 誕生日 | 必須チェック | 空文字の場合エラー           |
| 年齢  | 必須チェック | null または NaN の場合エラー |
| 年齢  | 範囲チェック | 0未満または150超の場合エラー    |
| 成績  | 必須チェック | null または NaN の場合エラー |
| 成績  | 範囲チェック | 0未満または100超の場合エラー    |
| クラス | 選択チェック | 0の場合エラー             |

## 8. エラー処理

入力チェックエラーがある場合は、フォーム下部にエラーメッセージを一覧表示する。
削除時は confirm を使用し、利用者に確認する。
更新対象が存在しない場合は alert を表示する。

## 9. 今後の拡張案

今後バックエンドと連携する場合は、以下のAPIを追加する。

| 処理     | メソッド   | URL                           |
| ------ | ------ | ----------------------------- |
| 学生一覧取得 | GET    | /api/students                 |
| 学生検索   | GET    | /api/students?studentName=xxx |
| 学生登録   | POST   | /api/students                 |
| 学生更新   | PUT    | /api/students/{studentId}     |
| 学生削除   | DELETE | /api/students/{studentId}     |
